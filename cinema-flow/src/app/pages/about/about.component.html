路由拓扑

/                  → 重定向到 /dashboard
/dashboard         → DashboardComponent（仪表盘）
/movies            → MovieListComponent（全部电影）
/movies/genre/:genre → MovieListComponent（分类筛选）
/movies/:id        → MovieDetailComponent（电影详情）
/directors         → DirectorListComponent（导演列表）
/directors/:id     → DirectorDetailComponent（导演详情）
/add               → MovieAddComponent（添加电影）
/about             → AboutComponent（关于）
/**                → 重定向到 /dashboard（404 兜底）

已注册服务

  • MovieService — 电影数据 Observable 接口
  • DirectorService — 导演数据 Observable 接口
  • MessageService — 业务消息日志
  • MovieStateService — BehaviorSubject 状态中心
